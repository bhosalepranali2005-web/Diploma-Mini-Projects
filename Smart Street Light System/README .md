# Smart Street Light System

## Description
The Smart Street Light System automatically turns street lights ON at night and OFF during daytime using an LDR sensor. This helps reduce electricity consumption and enables efficient energy usage.

## Components Used
- Arduino UNO
- LDR Sensor
- Relay Module
- LED / Street Lamp
- Resistor
- Power Supply

## Working Principle
The LDR sensor detects the light intensity of the environment. 
When the light level decreases during nighttime, the Arduino activates the relay which turns ON the street light. 
During daytime, the light intensity increases and the relay turns OFF the light automatically.

## Applications
- Smart cities
- Automatic street lighting
- Energy saving systems
