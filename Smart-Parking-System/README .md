# Smart Parking System

## Description

The Smart Parking System is designed to detect available parking spaces using sensors and display the status to users. The system helps drivers easily identify free parking slots, reducing time spent searching for parking and improving traffic management.

Sensors are placed in each parking slot to detect whether a vehicle is present. The Arduino reads sensor data and updates the parking availability using LEDs or an LCD display.

## Components Used

* Arduino UNO
* IR Sensors / Ultrasonic Sensors
* LEDs or LCD Display
* Resistors
* Connecting Wires
* Power Supply

## Working Principle

Each parking slot is equipped with a sensor that detects the presence of a vehicle.

The system works as follows:

1. Sensors detect whether a parking slot is occupied or empty.
2. The Arduino processes sensor signals.
3. LEDs or an LCD display indicate available or occupied parking spaces.

This allows drivers to quickly find empty parking slots.
