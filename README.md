# Diploma Mini Projects – Embedded & Arduino

This repository contains a collection of hardware-based mini projects developed during my Diploma in Electronics and Telecommunication Engineering (ENTC).

The projects focus on embedded systems, sensor interfacing, and automation using Arduino and basic electronic components.

## Projects Included

* Smart Street Light System
* Automatic Plant Watering System
* Digital Temperature Monitoring System
* Gas Leakage Detection System
* Smart Parking System
* Automatic Door Opening System
* Fire Detection Alarm System
* Water Level Indicator
* Bidirectional Visitor Counter

## Technologies Used

* Arduino
* Embedded C
* Sensors (LDR, IR, MQ2, LM35, Soil Moisture, Flame)
* Relay Modules
* LCD Displays
* Basic Electronics Components

## Purpose

These projects demonstrate practical applications of embedded systems, automation, and sensor-based monitoring systems commonly used in IoT and smart infrastructure.
