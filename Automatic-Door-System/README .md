# Automatic Door Opening System

## Description

The Automatic Door Opening System is designed to open a door automatically when a person approaches. The system uses an IR sensor to detect motion or presence and activates a motor or servo mechanism to open the door.

This project demonstrates basic automation using sensors and microcontrollers and is commonly used in shopping malls, hospitals, offices, and smart buildings.

## Components Used

* Arduino UNO
* IR Sensor Module
* Servo Motor
* Connecting Wires
* Power Supply

## Working Principle

The IR sensor continuously monitors the area in front of the door.

When a person comes near the sensor:

* The IR sensor detects the presence
* A signal is sent to the Arduino
* Arduino activates the servo motor
* The door automatically opens

After a short delay, the door closes automatically.
